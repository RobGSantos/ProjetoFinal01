Motivo: revisar conteúdos prévios do nível 1 antes de continuar com EF Core, Identity e JWT

## Objetivo do projeto

API Web desenvolvida em ASP.NET Core para estudo, com foco em gerenciamento de cursos, estudantes, autenticação e matrículas.

## Tecnologias utilizadas

Tecnologias principais:
- .NET 8
- ASP.NET Core Web API
- Controllers
- Swagger/OpenAPI
- Git/GitHub

## Requisitos para executar

- SDK .NET 8 ou superior
- Git
- Editor de código, como Visual Studio Code ou Visual Studio

## Como rodar localmente

1. Clonar o repositório.
2. Entrar na pasta do projeto.
3. Restaurar os pacotes.
4. Executar a API.
5. Acessar o Swagger pelo navegador.

## Endereço do Swagger

Ao rodar localmente, o Swagger estará disponível em uma URL parecida com:

https://localhost:7001/swagger

ou

http://localhost:5001/swagger

A porta pode variar conforme o `launchSettings.json`.
